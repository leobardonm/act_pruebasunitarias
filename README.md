# Clasificación de triángulos

Proyecto en JavaScript para clasificar triángulos como equiláteros, isósceles o escalenos, con pruebas unitarias usando Jest.

## Requisitos

- Node.js instalado
- Dependencias del proyecto instaladas con `npm install`

## Ejecutar pruebas

```bash
npm test
```

El comando ejecuta Jest en modo secuencial y valida todas las pruebas unitarias del proyecto.
